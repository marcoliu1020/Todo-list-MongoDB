# Todo-list-MongoDB

## description
todo list app with react, MongoDB

## create ./server/.env file
PORT=5500<br>
DB_CONNECT=<your db connection>

## start server
- open new terminal
- cd server
- npm start

## start react
- open new terminal
- cd client
- npm start
